# Add Localization Workflow

This workflow guides you through adding localization resources.

## Step 1: Add English String

Open `presentation/common/src/commonMain/composeResources/values/strings.xml`.
Add the string resource.

```xml
<string name="my_string">Content</string>
```

## Step 2: Add Japanese String

Open `presentation/common/src/commonMain/composeResources/values-ja/strings.xml`.
Add the translated string.

```xml
<string name="my_string">内容</string>
```

## Step 3: Use in UI

Use `stringResource` in your Composable.

```kotlin
Text(stringResource(Res.string.my_string))
```
